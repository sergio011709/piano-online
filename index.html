🎹 Piano Web Interactivo

Tócalo con el ratón, con el dedo o con el teclado del ordenador. Usa las teclas blancas y negras para tocar dos octavas completas con sonido generado en tiempo real.

Modo digital: ratón y táctil
Modo teclado: ordenador
Sin nota activa
Teclado — primera octava Z, X, C, V, B, N, M = teclas blancas
S, D, G, H, J = teclas negras
Teclado — segunda octava Q, W, E, R, T, Y, U = teclas blancas
2, 3, 5, 6, 7 = teclas negras
Consejo Si no suena al principio, pulsa cualquier tecla del piano o del teclado para activar el audio del navegador.
Hecho en una sola página HTML con Web Audio API, sin dependencias externas. Puedes guardar este archivo y abrirlo en cualquier navegador moderno.
